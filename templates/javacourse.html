What is Java?


Java is a versatile, high-level programming language designed to be platform-independent, meaning you can write code once and run it anywhere. Developed by Sun Microsystems and now maintained by Oracle, Java is widely used for building a variety of applications, from web and mobile apps to large-scale enterprise systems. Its object-oriented structure, robust security features, and extensive libraries make it a popular choice for developers around the world. Javaâ€™s "write once, run anywhere" philosophy is powered by the Java Virtual Machine (JVM), which allows Java applications to run on any device or operating system that has a JVM installed.


Module 1: Printing in Java


Concept: Printing in Java refers to the process of sending output to the console. This is essential for displaying information, debugging, and providing feedback to users.


  • System.out.print(): Prints text to the console without adding a newline at the end. This means subsequent prints will continue on the same line.
  • System.out.println(): Prints text to the console and adds a newline at the end. This ensures that the next output starts on a new line.
  • System.out.printf(): Prints formatted text. You can include placeholders in the string that are replaced with variable values. This method allows for precise control over the formatting of the output.

Example


System.out.print("Hello, ");
System.out.println("World!");


This will output Hello, World! on the same line because System.out.print does not append a newline, while System.out.println does.


Module 2: Different Data Types


Primitive Data Types


Concept: Primitive data types are the most basic data types available in Java. They represent simple values and are not objects.

  • int: Used to store integer values. It is a 32-bit signed integer. (Example: int age = 25;)
  • float: Used to store single-precision floating-point numbers (decimals). It is a 32-bit IEEE 754 floating point. (Example: float pi = 3.14f;)
  • double: Used to store double-precision floating-point numbers. It is a 64-bit IEEE 754 floating point and offers more precision than float. (Example: double e = 2.718281828;)
  • char: Used to store a single 16-bit Unicode character. (Example: char initial = 'A';)
  • boolean: Used to store true or false values (Example: boolean isJavaFun = true;)

Reference Data Types


Concept: Reference data types refer to objects and arrays. They hold references to data rather than the data itself.

  • String: Represents a sequence of characters. Strings are objects and offer many methods for manipulating text. (Example: String greeting = "Hello";)
  • Arrays: Used to store multiple values of the same type in a single variable. Arrays have a fixed size once created. (Example: int[] numbers = {1, 2, 3, 4, 5};)

String message = "Hello, Java!";
int[] numbers = {1, 2, 3, 4, 5};


message holds a reference to a String object, while numbers holds a reference to an array of integers.


Module 3: Conditionals


Concept: Conditionals allow your program to make decisions based on certain conditions. They execute different code blocks depending on whether a condition is true or false.


if Statement: Evaluates a condition and executes a block of code if the condition is true


if (condition) {
// Code to execute if condition is true
}


else if Statement: Used to test multiple conditions. Follows an if statement and provides an alternative condition to test.


if (condition1) {
// Code if condition1 is true
} else if (condition2) {
// Code if condition2 is true
}


else Statement: Executes a block of code if all preceding if and else if conditions are false.


if (condition) {
// Code if condition is true
} else {
// Code if condition is false
}


int x = 3;
if(x == 3){
System.out.println("True");
}
else if(x == 2){
// Nothing occurs
}
else{
// If no other conditionals are met:
System.out.println("All failed");
}


switch Statement: Allows for the selection of one of many code blocks to execute. Useful when multiple conditions are based on the same variable.


switch (variable) {
case value1:
// Code if variable equals value1
break;
case value2:
// Code if variable equals value2
break;
default:
// Code if variable does not match any case
}


int day = 3;
switch (day) {
case 1:
System.out.println("Monday");
break;
case 2:
System.out.println("Tuesday");
break;
case 3:
System.out.println("Wednesday");
break;
default:
System.out.println("Invalid day");
break;
}


This code will print "Wednesday" because day is equal to 3.


Module 4: Loops


Concept: Loops allow you to repeatedly execute a block of code as long as a specified condition is true. They are used for tasks that require repetition.


for Loop: Ideal when the number of iterations is known beforehand. Includes initialization, condition, and iteration expression.


for (initialization; condition; iteration) {
// Code to execute on each iteration
}


while Loop: Executes a block of code as long as the specified condition is true. Suitable when the number of iterations is not known in advance.

while (condition) {
// Code to execute while condition is true
}

do-while Loop: Similar to while loop, but guarantees that the code block is executed at least once before checking the condition.


do {
// Code to execute
} while (condition);


for (int i = 1; i <= 5; i++) {
System.out.println("Iteration: " + i);
}
int i = 1;
while (i <= 5) {
System.out.println("Iteration: " + i);
i++;
}
int j = 1;
do {
System.out.println("Iteration: " + j);
j++;
} while (j <= 5);


The for loop and while loop will output numbers from 1 to 5. The do-while loop will do the same but guarantees at least one iteration.


Key Points Summary


  • Printing: Output text to the console using print(), println(), and printf().
  • Data Types: Understand and use primitive (e.g., int, float) and reference types (e.g., String, arrays).
  • Conditionals: Implement logic with if, else if, else, and switch statements.
  • Loops: Repeat code execution using for, while, and do-while loops.
