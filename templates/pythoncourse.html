What is Python?



Python is a high-level, interpreted programming language known for its simplicity and readability. Designed to be easy to learn and use, Python emphasizes code clarity and reduces the cost of program maintenance. Its versatile nature makes it ideal for a wide range of applications, from web development and data analysis to artificial intelligence and scientific computing. With a rich ecosystem of libraries and frameworks, such as Django, Flask, and TensorFlow, Python empowers developers to build robust applications and perform complex computations efficiently. Printing



  • The print() function is used to display information to the console. You can print strings, numbers, and other data types.
    • Printing Multiple Items
  • You can print several items separated by commas. The print() function will insert spaces between them.
    • Formatting Strings
  • You can format strings to include variables and other data. This is useful for creating user-friendly output.

Basic Print Statement:


# Print a simple string
print("Hello, World!") # Output: Hello, World!


Printing Multiple Items:

# Print multiple items with a space between them
print("Hello,", "World!") # Output: Hello, World!


String Formatting:


name = "Alice"
age = 30
# Using f-strings for formatting
print(f"Name: {name}, Age: {age}") # Output: Name: Alice, Age: 30
# Using .format() method for formatting
print("Name: {}, Age: {}".format(name, age)) # Output: Name: Alice, Age: 30



Data Types


  • Integers (int): Whole numbers, e.g., 1, 100, -7.
  • Floats (float): Numbers with decimal points, e.g., 3.14, -0.001.
  • String (str): Text data enclosed in quotes, e.g., "hello", 'world'.
  • Boolean (bool): Represents True or False.
  • Lists (list):Ordered, mutable collections of items. Items can be of different types, e.g., [1, "hello", 3.14].

Conditionals


if Statements: Check if a condition is true and execute code based on that. elif (Else If) Statements: Check additional conditions if the initial if condition is false. else Statement: Execute code when all preceding conditions are false

Comparison Operators:

  • (==): (equal to)
  • (!=): (not equal to)
  • (>): (greater than)
  • (<): (less than)
  • (>=): (greater than or equal to)
  • (<=): (less than or equal to)

Basic Conditionals:

age = 18
if age < 18:
   print("You are a minor.")
elif age == 18:
   print("You are an adult now!")
else:
   print("You are an adult.")


Comparison Operators:

num1 = 10
num2 = 20
print(num1 == num2) # Output: False
print(num1 < num2) # Output: True
print(num1 != num2) # Output: True



Loops

  • for Loops: Iterate over a sequence (e.g., list, string) and execute code for each item.
  • while Loops: Repeat code as long as a condition remains true.
  • range() Function: Generates a sequence of numbers, often used in for loops.
  • break Statement: Exit the loop prematurely.
  • continue Statement: Skip the current iteration and continue with the next one.

For Loop:

# Iterating over a range
for i in range(5):
   print(i) # Output: 0 1 2 3 4


While Loop:

count = 0
while count < 5:
   print(count) # Output: 0 1 2 3 4
   count += 1


Break and Continue:

for i in range(10):
if i == 5:
   break # Exit loop when i equals 5
if i % 2 == 0:
   continue # Skip even numbers
print(i) # Output: 1 3
