What is Javascript?

Module 1: Printing in JavaScript

Introduction to Printing

Concept: Printing in JavaScript refers to outputting information to the web page or console. This is essential for displaying information, debugging, and providing feedback.

  • console.log(): Prints text to the browser's console. Useful for debugging and logging information.
  • alert(): Displays a dialog box with a message. Useful for simple alerts and notifications.
  • document.write(): Writes text directly to the HTML document. Generally used for simple demonstrations or debugging.
console.log("Hello, World!"); // Outputs to the console alert("Hello, World!"); // Displays a popup alert document.write("Hello, World!"); // Writes directly to the HTML document

Module 2: Different Data Types

Primitive Data Types

Concept: Primitive data types represent simple values. JavaScript has several primitive types, including:

  • number: Represents both integer and floating-point numbers. (Example: let age = 30;)
  • string: Represents a sequence of characters enclosed in single quotes, double quotes, or backticks (template literals). (Example: let name = "Alice";)
  • boolean: Represents true or false values. (Example: let isJavaScriptFun = true;)
  • undefined: Represents a variable that has been declared but not yet assigned a value. Example: let notDefined;
  • null: Represents the intentional absence of any value. (Example: let emptyValue = null;)
  • symbol (ES6): Represents a unique and immutable value. (Example: let uniqueSymbol = Symbol('description');)

Reference Data Types

Concept: Reference data types refer to objects and arrays. They hold references to the data rather than the data itself.


object: Represents a collection of key-value pairs. Objects can contain properties and methods.

let person = { name: "Alice", age: 30 };

array: Represents a list-like collection of values. Arrays can hold values of different types and are zero-indexed.

let numbers = [1, 2, 3, 4, 5];


let message = "Hello, JavaScript!"; let numbers = [1, 2, 3, 4, 5]; let person = { name: "Alice", age: 30 }; console.log(message); // Outputs: Hello, JavaScript! console.log(numbers); // Outputs: [1, 2, 3, 4, 5] console.log(person); // Outputs: { name: 'Alice', age: 30 }



Module 3: Conditionals

Introduction to Conditionals

Concept: Conditionals allow your code to make decisions based on certain conditions. They execute different code blocks depending on whether a condition is true or false.

if Statement: Executes a block of code if the specified condition is true.

if (condition) { // Code to execute if condition is true }


else if Statement: Follows an if statement to test additional conditions.

if (condition1) { // Code if condition1 is true } else if (condition2) { // Code if condition2 is true }


else Statement: Executes a block of code if all preceding conditions are false.

if (condition) { // Code if condition is true } else { // Code if condition is false }

switch Statement: Allows for the selection of one of many code blocks to execute based on the value of a variable.

switch (variable) { case value1: // Code if variable equals value1 break; case value2: // Code if variable equals value2 break; default: // Code if variable does not match any case }



let day = 3; switch (day) { case 1: console.log("Monday"); break; case 2: console.log("Tuesday"); break; case 3: console.log("Wednesday"); break; default: console.log("Invalid day"); break; }


This code will output "Wednesday" because day is equal to 3.




Module 4: Loops

Introduction to Loops

Concept: Loops allow you to execute a block of code repeatedly as long as a specified condition is true. They are used for tasks that require repetition.

for Loop: Ideal when the number of iterations is known beforehand. It includes initialization, condition, and iteration expression.

for (initialization; condition; iteration) { // Code to execute on each iteration }

while Loop: Executes a block of code as long as the specified condition is true. Suitable when the number of iterations is not known in advance.

while (condition) { // Code to execute while condition is true }

do-while Loop: Similar to while loop, but guarantees that the code block is executed at least once before checking the condition.

do { // Code to execute } while (condition);

for (let i = 1; i <= 5; i++) { console.log("Iteration: " + i); } let i = 1; while (i <= 5) { console.log("Iteration: " + i); i++; } let j = 1; do { console.log("Iteration: " + j); j++; } while (j <= 5);

The for loop and while loop will output numbers from 1 to 5. The do-while loop will do the same but guarantees at least one iteration.




Summary of Key Points

  • Printing: Output text to the console or web page using console.log(), alert(), and document.write().
  • Data Types: Use primitive types (e.g., number, string) and reference types (e.g., object, array).
  • Conditionals: Implement decision-making with if, else if, else, and switch statements.
  • Loops: Repeat code execution using for, while, and do-while loops.
